1. Проект на базе Gulp. Команды: 'gulp' - режим редактирования проекта, 'gulp sprite' - компоновка svg sprite, 'gulp build' - собрать проект. Папка 'app' - рабочий проект, папка 'dist' - готовая сборка для использования, файлы 'gulpfile.js' - настройки gulp и 'package.json' - плагины gulp.
Для работы с gulp нужно установить Node.JS (настройки по умолчанию), среда - VS Code.
Развернуть проект - 'npm install'

2. Html страницы объединены в SPA. Меньше кода, единый header для всех, повышение скорости работы. Навигации осуществляется путём показа/скрытия блоков c тегом 'section', базовый класс 'page'. Список блоков 'page-feeding,' 'page-schedule', 'page-settings'. Модификатор класса 'page-show' - показывает блок, для скрытия нужно удалить класс.

3. Svg картинки кота оптимизированы и представлены кодом в теле html.

4. Повторяемые svg элементы оптимизированы в svg sprite. Пример вызова элемента спрайта - <svg class="bars__icon"><use xlink:href="images/sprite.svg#bars"></use></svg>. Путь можно вставлять в стили, тег 'img' и тег 'svg'. Список элементов можно найти в app/images/stack/sprite.stack.html

5. Раздел настройки, аккардион, текстовое поле вывода информации. Тег 'p', базовый клас 'textinfo'. Модификаторы класса 'textinfo-success' - зеленый, 'textinfo-alert' - желтый, 'textinfo-error' - красный, 'textinfo-action' - серый, 'textinfo-hidden' - скрыть элемент.

6. Favicon упрощен до одного png файла с разрешением 32x32. Это наиболее используемая практика, открывается на всех устройствах и занимает меньше памяти.

7. Верстка максимально с PerfectPixel. Но в некоторых места оптимизирована в угоду адаптивности. Раздел Покормить, немного изменены отступы элементов. Так же в некоторых местах исправлены padding на margin и наоборот.

8. Убраны bootstrap, JQuery и icomoon. Исправлены практики и теги bootstrap на более правильные и оптимизированные. JQuery переписан на чистый JS, icomoom заменен на svg sprite, технология шрифтов устаревшая и больше потребляет памяти, так же не на всех устройствах могут работать правильно.

9. Иконки немного отличаются. Взяты из свободно распространяемых - FontAwesome. Иконка bars изменена на svg код.

10. Кнопки. Базовый класс 'button'. Модификаторы цвета: 'button-green', 'button-blue', 'button-cyan'. Модификаторы позиций: 'button-left', 'button-center', 'button-right'. Модификатор активности: 'button-disabled' (для неактивности кнопки дополнительно нужно добавить в html код - атрибут 'disabled').
Кнопки с иконками реализована теми же кнопками, но с добавлением в них элементов svg sprite. Пример: <button class="button button-blue button-right"><svg class="button-icon"><use xlink:href="images/sprite.svg#arrows-rotate"></use></svg>Выполнить поиск Wi-Fi сетей</button> . Класс 'button-icon_rotation' добавляет иконке кнопки вращение.

12. Раздел настройки, wi-fi панель. Для показа блока нужно добавить класс 'wifipanel_show', скрыть - убрать класс. Уровень сигнала wi-fi - svg картинки, перерисованы по скрину. Выбор элемента списка wi-fi реализован добавлением класса 'wifipanel__card-target', цвет выделения там же задается. Пример элемента:
